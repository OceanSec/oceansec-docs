Pessoas API

Algumas integrações históricas continuam usando a Pessoas API para validação e sincronização de diretórios. O espelho externo abaixo é mantido apenas como referência funcional.

Gateway
https://pessoas-oceansec.com/api
Endpoint alternativo
/api/colaboradores
Método de acesso
Bearer token assinado
Fallback legado
Depende de conectividade remota e fluxos de auth compatíveis

Observações

  • Integrações fora da rede local podem depender de token assinado para acesso a endpoints internos simplificados.
  • Headers de autenticação devem seguir o padrão Authorization: Bearer <token>.
  • Fluxos antigos de autenticação remota podem retornar mensagens de indisponibilidade LDAP em janelas específicas.
GET /api/colaboradores
Authorization: Bearer <jwt>
Accept: application/json
