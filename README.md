# OceanSec Engineering Docs

Portal de documentação técnica da OceanSec, contendo referências operacionais, notas de arquitetura e guias de integração utilizados por equipes internas e parceiros.

Este repositório publica uma versão estática do conteúdo, com foco em:

- Identity & Access (SSO, LDAP, VPN)
- Network & Edge (proxy, DNS, conectividade)
- Infraestrutura e serviços legados
- Cloud & Storage (S3, retenção, fluxos de incidente)
- APIs internas e integrações
- Runbooks operacionais

O objetivo é fornecer uma base leve de consulta para troubleshooting, onboarding técnico e suporte a integrações externas.
